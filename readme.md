# Parts
## Required
- Raspberry Pi 4
- 5v DC Power Supply

## Optional
- HDMI Screen
- Motion Sensor

# Setup
### 1. Flash the sd card
1. Open Raspberry Pi Imager
    - Operating System: Raspberry Pi OS (64-bit)
    - Hostname: ariahub

### 2. Run the setup script
```shell
sudo chmod +x setup.sh
sudo ./setup.sh
```
